# TransactionButton

`src/components/atoms/TransactionButton/`

Botão de ação de transação: repassa atributos nativos de `<button>` (via `ButtonHTMLAttributes`) e aplica a classe base do módulo SCSS.

## Props

| Prop | Tipo | Descrição |
|------|------|-----------|
| `children` | `React.ReactNode` | Conteúdo |
| `className` | `string` | Classe extra |
| `...` | atributos de `button` | `onClick`, `type`, `disabled`, etc. |
