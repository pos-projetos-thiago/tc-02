# Catálogo de componentes

Tudo o que está em `src/components/` com página própria nesta documentação. Use o menu **Componentes** (agrupado em Átomos, Moléculas, Organismos e Templates) para abrir cada uma.

A página do [Button](./button) traz o exemplo mais longo (variantes e trechos de SCSS); as demais seguem o mesmo padrão: caminho do código, descrição curta e tabela de props extraída do TypeScript.

## Visão por pasta

| Camada | Onde no menu |
|--------|----------------|
| `atoms/` | **Átomos** |
| `molecules/` | **Moléculas** |
| `organisms/` | **Organismos** |
| `templates/` | **Templates** |
