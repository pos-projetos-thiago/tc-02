# DashboardNav

`src/components/molecules/DashboardNav/`

Navegação do **dashboard**: em desktop, `aside` com links para as seções; em faixa de tablet, `Tabs` do MUI. Usa `useDashboard()` de `DashboardContext` para `activeSection` e `setActiveSection`.

Seções: Início (`services`), Transferências, Investimentos, Outros serviços.

## Props

Nenhuma. Comportamento totalmente acoplado ao contexto.
